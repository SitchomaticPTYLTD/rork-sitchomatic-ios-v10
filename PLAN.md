# PPSR CarCheck App

## Overview

PPSR-only card testing application with VIN & card validation against the Australian PPSR (Personal Property Securities Register).

## Features

- **PPSR Card Testing** — Automated VIN + card validation via the PPSR CarCheck portal
- **Concurrent Sessions** — Up to 8 simultaneous PPSR checks
- **NordVPN Config Generator** — WireGuard & OpenVPN config generation via NordLynx API
- **IP Score Testing** — 8x concurrent IP reputation checks
- **Network Infrastructure** — DNS-over-HTTPS, SOCKS5 proxy, WireGuard, OpenVPN support
- **Debug Logging** — Comprehensive diagnostic logging with session tracking
- **Persistent Vault** — Full app state backup and restore

## Screens

1. **Main Menu** — PPSR, Nord Config, IP Score, Debug Log, Vault

3. **PPSR Settings** — Email rotation, stealth, network configuration
4. **Nord Config** — VPN config generation and management
5. **IP Score Test** — Concurrent IP reputation testing
6. **Debug Log** — Full diagnostic log viewer
7. **Vault** — File storage browser and backup management
